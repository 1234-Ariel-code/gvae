# GitHub setup guide

## Suggested repository name
`gvae-genomic-variational-autoencoder`

## Suggested top-level message

This repository contains the public software release for the gVAE manuscript. It includes a data-safe synthetic demo, the method implementation, figure-generation scripts, and an interactive Streamlit application.

## Recommended GitHub sections

### About
- Stable representation learning for high-dimensional genomic data
- Synthetic demo included
- Streamlit app included

### Topics
`genomics`, `variational-autoencoder`, `representation-learning`, `bioinformatics`, `snps`, `streamlit`, `xai`
