Synthetic demo data are generated locally by `scripts/generate_synthetic_data.py`.

Each generated bundle includes:

- `genotypes.csv` and `phenotype.csv`
- `demo_filtered.csv`
- `demo_origin.phen`, `demo_origin.tped`, `demo_origin.tfam`
- `demo.bed`, `demo.bim`, `demo.fam`
- `demo_gwas.assoc` or `demo_gwas.qassoc`

We run either the lightweight public demo or a synthetic version of the original PLINK-based pipeline.
